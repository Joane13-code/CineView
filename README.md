# CineView
O CineView é um site de catálogo de filmes que permite visualizar resenhas, trailers e informações sobre diversos filmes famosos. Também oferece páginas para login e cadastro de usuários. O projeto foca em design responsivo, uso de HTML e CSS com visual moderno e estilizado.
